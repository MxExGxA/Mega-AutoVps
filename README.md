* Squid Proxy Server
* Openvpn Server 
* V2ray Server(Soon)
* SHH Server(Soon)
* SSL (Soon)
### Tested on:
* Debian 9
* Debian 10
* Debian 11
## Installation
apt install git -y && git clone https://github.com/MxExGxA/Mega-AutoVps && chmod +x install.sh && ./install.sh
