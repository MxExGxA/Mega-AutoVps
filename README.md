# Mega-AutoVps
### This Script Will Install:
* Squid Proxy Server
* Openvpn Server 
* V2ray Server(Soon)
* SSH DropBear Server(Soon)
* SSL (Soon)

## Only Supports Ubuntu & Debian
### Tested on:
* Debian 9,10,11
* Ubuntu 18,20
## Installation
apt install git -y && git clone https://github.com/MxExGxA/Mega-AutoVps && cd Mega-AutoVps && chmod +x install.sh && ./install.sh
